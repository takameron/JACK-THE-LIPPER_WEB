JACK THE LIPPER API

仕様

登録

メソッド HTTPリクエスト 概要
POST /api/v1/dryness 新規登録

パラメータ

名前 必須か 項目 注釈 例
lat 必須 緯度 36.111221
long 必須 経度 137.950266
score 必須 乾燥度 1（最低）~5（最高）の5段階 3
age 任意 年齢 20
sex 任意 性別 0:女性 1:男性 0

リクエスト例
http://127.0.0.1:3000/api/v1/dryness?lat=36.111221&long=137.950266&age=20&sex=0&score=3

パラメータ

名前 必須か 項目 注釈 例

リクエスト例

パラメータ

APIの動作確認

PythonのRequestsライブラリを使用すると、任意のHTTPメソッドでURLにアクセスできます。

動作例
$ pip install requests
$ python3
>>> import requests
>>> requests.post('http://127.0.0.1:3000/api/v1/dryness?lat=36.111221&long=137.950266&age=20&sex=0&score=3')
>>> requests.get('http://127.0.0.1:3000/api/v1/dryness/a97768ae-4bdc-4554-8d05-417e90f81264')
>>> requests.put('http://127.0.0.1:3000/api/v1/dryness/a97768ae-4bdc-4554-8d05-417e90f81264?lat=36.111221&long=137.950266&age=50&sex=1&score=2')
>>> requests.delete('http://127.0.0.1:3000/api/v1/dryness/a97768ae-4bdc-4554-8d05-417e90f81264')
